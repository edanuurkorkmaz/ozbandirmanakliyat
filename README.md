# Öz Bandırma Nakliyat – Kurumsal Website

Bu proje, Öz Bandırma Nakliyat firması için geliştirilmiş modern ve responsive kurumsal web sitesidir.

Uygulama React ve Vite altyapısı ile geliştirilmiş olup; performans, sürdürülebilirlik ve SEO uyumluluğu ön planda tutularak tasarlanmıştır.

---

## Proje Amacı

- Kurumsal kimliği dijital ortamda güçlü şekilde yansıtmak
- Hizmetleri açık ve anlaşılır biçimde sunmak
- Mobil uyumlu ve hızlı bir kullanıcı deneyimi sağlamak
- SEO uyumlu teknik altyapı oluşturmak
- Ölçeklenebilir ve sürdürülebilir bir frontend mimarisi kurmak

---

## Kullanılan Teknolojiler

- React
- Vite
- Tailwind CSS
- React Router
- Responsive Design
- SEO uyumlu yapı

---

## Kurulum

Projeyi lokal ortamda çalıştırmak için:

```bash
# Repoyu klonla
git clone https://github.com/kullaniciadi/ozbandirmanakliyat.git

# Proje klasörüne gir
cd ozbandirmanakliyat

# Bağımlılıkları yükle
npm install

# Geliştirme sunucusunu başlat
npm run dev
```
